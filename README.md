## 📌 Sobre o Projeto

O projeto foi desenvolvido para a disciplina de Segurança da Informação com o objetivo de criar uma ferramenta capaz de realizar diagnósticos de conformidade organizacional utilizando controles das normas internacionais de segurança da informação.

A aplicação permite registrar auditorias, avaliar controles, armazenar históricos e apresentar dashboards analíticos com indicadores de conformidade.

### 🎯 Objetivos do Sistema

- ✅ Avaliar conformidade baseada na ISO 27001
- ✅ Utilizar controles da ISO 27002 para diagnóstico
- ✅ Implementar módulo específico para ISO 27701
- ✅ Gerar dashboards e gráficos de conformidade
- ✅ Armazenar histórico de auditorias
- ✅ Permitir comparações evolutivas entre diagnósticos
- ✅ Gerar relatórios completos e parciais

### 🏗️ Arquitetura UML

Modelagem geral da ferramenta e interação entre módulos.

### 🔄 Diagrama de Casos de Uso

Fluxos principais:

- Realizar auditoria
- Gerar relatório
- Visualizar dashboard
- Comparar auditorias

### 🔍 Diagnóstico de Controles

Para cada controle da ISO 27002 o auditor poderá selecionar um dos seguintes status:

- ✅ Conforme → Controle implementado corretamente
- ❌ Não Conforme → Controle não implementado ou inadequado
- ⚪ Não se Aplica → Controle não aplicável ao cenário avaliado
